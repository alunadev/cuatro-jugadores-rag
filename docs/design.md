# Design System — CuatroJugadoresAI

## Identity
- Marca basada en técnica + cercanía + entrenamiento.
- Colores principales:
  - Verde #4CAF50 (progreso, aprendizaje)
  - Gris oscuro #1A1A1A (seriedad)
  - Blanco #FFFFFF (lectura)

## Typography
- Headings: Inter Bold
- Body: Inter Regular
- Code blocks: JetBrains Mono

## Layout
- Chat panel en 2 columnas
- Panel lateral con acceso a:
  - Crear edición
  - Buscar consejos
  - Mis ediciones generadas

## Components
- Markdown Editor
- Citations Tooltip
- "4 Points Cards"
- Footer fijo con CTA “Siempre nos falta un cuarto”
